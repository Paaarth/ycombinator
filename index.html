About HN Reader

A Hacker News client built with Bootstrap and JavaScript that uses the official HN API.

Features:

  • View top/new/best stories
  • Read Ask HN, Show HN, and Job posts
  • Browse comments and user profiles
  • Real-time updates
  • Search functionality
